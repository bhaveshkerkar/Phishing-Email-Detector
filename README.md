🔍 Overview

This project uses Machine Learning to detect whether an email is Phishing or Legitimate, based on its content.
Developed as part of the Digisuraksha Cybersecurity Internship (2025).

🧠 Features

✨ Machine Learning–based classification
✨ Email preprocessing & TF-IDF vectorization
✨ Easy-to-run Python script
✨ Lightweight, fast, and accurate
✨ Detects phishing patterns using text analysis
